# MicroC
Pipeline to analysis MicroC data.

## Usage

1. Copy `config_template.sh` to `config.sh` and edit paths
2. Source the config: `source config.sh`
3. Modify sample list in `run_microc_pipeline.sh`
4. Run: `bash run_microc_pipeline.sh`

## Dependencies

- bwa
- samtools
- pairtools
- Juicer Tools
- Java 8+
